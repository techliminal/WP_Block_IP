== Description ==
Set a block on IP's or a range of IP's.

Once activated, Go to Settings -> IP Block.

To enable on a page or post, just add a [block_ip] at the top (or bottom) of the content. 

Notes:

This plugin is based on Best WP Security plugin by flippercode.  
Link: http://wordpress.org/support/view/plugin-reviews/best-wp-security


== Installation ==
This section describes how to install the plugin and get it working.
	
	1.	Upload the best-wp-security directory to the /wp-content/plugins/ directory

	2.	Activate the plugin through the 'Plugins' menu in WordPress
